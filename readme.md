## Documentation

Tech Used:
    Node.js
    Express
    MongoDB
    Pug
    Postman
    MongoDB Compass

FEATURES:
    REST API
    Router Mounting
    Environment variables
    CRUD
    MVC architecture
    Filtering(Basic, advanced: gte|lte)
    Sorting
    Field Limiting
    Pagination
    Route Aliasing
    Password encryption
    JWT
    Route Protection
    Forgot Password
    Password Reset
    Sending emails with Nodemailer
    cookies
    Signup
    Login
    Logout
    Mailtrap
    Payments using stripe
    Checkout
    Responsive 
## Installation

Install my-project with npm

```bash
  npm install 
  npm start
```
    
## Tech Stack

**Client:** Javascript, CSS, Pug

**Server:** Node, Express, MongoDB
