 A showcase for old and new Cars. In this project user can select the old and new cars. It also have the functionality to filter cars according to price range. Below are the screenshots of the running project

## Features

* MVC Architecture
* Filtering
* Sorting
* Field Limiting
* Pagination
* Aliasing
* Data validation
* JWT authentication
* Route protection


## Tech Stack

**Server:** Node, Express

**Database:** MongoDB

**Client:** Pug, CSS
